## ğŸ“¦ Installation

To install the application, follow these steps:

1. Clone the repository:

```bash
git clone https://github.com/IkarosKurtz/Grizzly-Subscription.git
```

2. Install the dependencies:

```bash
npm install
```

3. Run the application:

